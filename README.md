# Skill Survivor Prototype

## 프로젝트 목적

Unity 6 환경에서 2D 탑다운 액션 로그라이트 게임을 제작하며, Unity 클라이언트 개발자로서 필요한 전투 시스템, UI 연동, 데이터 구조, 상태 관리, 저장/로드, 최적화 기초를 학습하고 포트폴리오로 정리하는 것을 목표로 한다.

## 장르

2D 탑다운 액션 로그라이트

## 핵심 루프

스테이지 입장  
→ 적 웨이브 등장  
→ 플레이어 이동 및 전투  
→ 경험치 획득  
→ 레벨업 시 스킬 선택  
→ 보스 등장  
→ 클리어 또는 실패 결과 출력

## 1차 목표

- 플레이어 이동
- 적 추적
- 기본 공격
- 체력 UI
- 경험치/레벨업 구조
- 스킬 선택 UI
- 결과 화면
