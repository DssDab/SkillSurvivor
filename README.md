# Skill Survivor Prototype

《Do it! 유니티 6로 게임 제작 & 출시하기》의 예제를 따라 구현하고, 학습한 기능을 하나의 프로젝트에 적용한 Unity 6 기반 2D 탑다운 액션 프로젝트입니다.

> 이 프로젝트는 창작 프로젝트가 아니라 Unity 기능과 게임 시스템 구현 흐름을 학습하기 위한 예제 기반 프로젝트입니다.

## 프로젝트 개요

- 장르: 2D 탑다운 액션 로그라이트
- 개발 환경: Unity 6, C#
- 개발 목적: Unity 게임 시스템의 구조와 데이터 흐름 학습
- 개발 상태: 예제 학습 및 프로젝트 적용 완료

## 주요 구현 기능

### 전투 및 스킬

- 스탯 기반 체력, 공격력, 치명타, 회피 처리
- 직선, 유도, 지속형, 광역 투사체 구현
- 버프, 회복, 회피 및 속성 시너지 구현
- 공격 결과를 적중, 회피, 무시 상태로 구분
- 회피 시 검정색 `Miss`, 적중 시 데미지 수치 출력

### 성장 및 UI

- 경험치 누적과 레벨업 처리
- 레벨업 시 무작위 스킬 선택
- 스킬 습득 및 레벨 증가
- 체력, 경험치, 보유 스킬 UI 갱신
- 일시정지와 로비 UI 구성

### 적과 스테이지

- NavMesh 기반 적 이동과 경로 탐색
- Behavior Tree 기반 적 상태 제어
- 적 생성 위치 안내와 웨이브 진행
- 스테이지 클리어 및 챕터 진행

### 데이터 및 편의 기능

- ScriptableObject 기반 스탯과 스킬 데이터 관리
- 메모리 풀을 활용한 오브젝트 관리
- 비동기 씬 전환
- JSON 직렬화를 활용한 저장 및 불러오기
- 모바일 가상 컨트롤러
- Android 빌드 환경 구성

## 주요 학습 내용

- 데이터와 런타임 객체의 역할 분리
- 이벤트를 활용한 게임 상태와 UI 갱신
- 투사체 유형에 따른 이동 및 충돌 처리
- 행동 트리를 활용한 적 행동 분기
- 오브젝트 생명주기와 사망 처리
- 비동기 씬 전환과 JSON 데이터 저장 흐름

## 주요 문제 해결

- 비활성화된 풀 오브젝트 참조로 발생한 오류 수정
- 제거된 적이 목록에 남아 발생하는 참조 문제 처리
- 누적 경험치에 따라 여러 레벨이 상승하도록 반복 계산 적용
- 공격 결과를 구분해 회피 및 사망 대상의 피격 피드백 처리

## 실행 및 테스트

- Unity 에디터에서 주요 기능 동작 확인
- Android 빌드 환경 구성 및 빌드 확인
- Android 실제 기기 조작 테스트는 진행하지 않음
